# 🛒 Blinkit Sales Dashboard

## 📌 Project Overview

The **Blinkit Sales Dashboard** is an interactive Power BI project designed to analyze sales performance, customer ratings, outlet distribution, and product categories. The dashboard provides business insights through KPIs, charts, and slicers, enabling users to explore sales trends and make data-driven decisions.

---

## 📷 Dashboard Preview

![Blinkit Dashboard](Blinkit%20Dashboard%202.png)

---

## 🎯 Objectives

- Analyze overall sales performance.
- Monitor key business KPIs.
- Compare sales across outlet types and locations.
- Identify top-performing product categories.
- Understand customer preferences based on fat content.
- Track outlet establishment trends over time.

---

## 📊 Key Performance Indicators (KPIs)

- 💰 **Total Sales:** $1.20M
- 💵 **Average Sales:** $141
- 📦 **Total Items Sold:** 8,523
- ⭐ **Average Rating:** 3.92

## 📂 Dataset

The dashboard is built using Blinkit sales data containing:

- Item Details
- Item Type
- Sales
- Ratings
- Outlet Type
- Outlet Size
- Outlet Location
- Establishment Year
- Fat Content

---

## 💡 Key Insights

- Supermarket Type 1 generated the highest total sales.
- Tier 3 outlets contributed the highest revenue.
- Low Fat products accounted for a significant share of total sales.
- Fruits & Vegetables and Snack Foods were the top-selling product categories.
- Outlet sales showed strong growth after 2018.
